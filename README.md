# 🎵 Mood Based Music Recommender

A simple web application that recommends and plays songs based on the user's mood.
Users can choose a mood and instantly see songs that match that feeling.

## 🚀 Features

* 🎧 Mood based song categories (Happy, Sad, Motivated, Chill)
* 🔍 Search songs by name or artist
* 🎲 Surprise Me button to play a random song
* 🎵 Built-in audio player
* 💡 Clean and interactive UI
* 📱 Responsive design

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## 🎮 How to Use

1. Open the website.
2. Select a mood (Happy, Sad, Motivated, Chill).
3. Click on any song card to play the song.
4. Use the search bar to find songs quickly.
5. Click **Surprise Me** to play a random song.
